PDF Generator API for Odoo

Design and generate custom PDFs from any Odoo record — no QWeb required

This module plugs pdfgeneratorapi.com into Odoo: design templates with a visual drag-and-drop editor, map Odoo fields to template placeholders without code, and generate PDFs from invoices, quotations, purchase orders, transfers, manufacturing orders, and rental orders.

How it is organised

This is the framework module: it ships the API client, the template editor embed, per-model field datasets, coverage diagnostics, and the async job machinery. It adds no buttons to your documents by itself — you pick the document types you want under Settings → PDF Generator API → Document types, and Odoo installs the matching bridge module automatically:

Document type Bridge module What it adds
Customer invoices & credit notes pdfgeneratorapi_connector_account Generate button on invoices, plus a Use pdfgen PDF toggle in the Send wizard so customers receive your design instead of the standard report
Quotations & sale orders pdfgeneratorapi_connector_sale Generate button + starter dataset for sale.order
Purchase orders pdfgeneratorapi_connector_purchase Generate button + starter dataset for purchase.order
Delivery slips & receipts pdfgeneratorapi_connector_stock Generate button + starter dataset for stock.picking
Manufacturing orders pdfgeneratorapi_connector_mrp Generate button + starter dataset for mrp.production
Rental orders pdfgeneratorapi_connector_rental Rental-specific dataset on top of the Sales bridge. Requires Odoo's Rental app (sale_renting, shipped with Odoo Enterprise)

The same machinery extends to any other Odoo model: create a field dataset for the model and inherit the document mixin — no hardcoded document types anywhere.

Key features

Drag-and-drop template editor

Open the pdfgeneratorapi.com editor inside Odoo. Design templates visually — no QWeb, no XML overrides, no developer required. Preview against a real Odoo record while you design.

Smart field mapping

Per-model datasets map Odoo fields to template placeholders, including repeated sections (order lines, taxes) and expressions composing several fields. Editable from the UI with a drag-and-drop field palette — no code.

Send-by-email integration

With the Invoicing bridge installed, the Send wizard can substitute the standard Odoo report with your latest generated PDF — customers receive your custom design, not the default invoice.

Async batch dispatch

Select hundreds of records in a list view and generate PDFs for all of them in one click. Per-record async jobs with a signed webhook callback keep Odoo responsive while pdfgeneratorapi.com renders in the background.

Coverage diagnostics

A built-in wizard inspects your template's placeholders and reports which ones are mapped, unmapped, or unused — catch broken templates before your customers do.

Multi-company & secure

Per-company API credentials with a global fallback, JWT-signed requests (HS256), webhook signature verification, and secret redaction in logs. An audit message is posted to the record's chatter on every generation.

How it works

1. Connect your workspace

Paste your pdfgeneratorapi.com API key, secret, and workspace identifier in Settings → PDF Generator API. Click Test Connection to verify.

2. Pick your document types

Tick the document types you need under Document types. Saving installs the matching bridge and seeds a ready-to-edit field dataset.

3. Design or import a template

Open the visual editor inside Odoo. Drag fields from the palette, style headers, add line tables, and preview against a real record.

4. Generate and send

Click Generate custom PDF on any supported record — the PDF is attached to the record and logged in the chatter, ready to download or send.

Configuration

  1. Sign up at pdfgeneratorapi.com and copy your API key, API secret, and workspace identifier (usually your account email). This module requires a pdfgeneratorapi.com account — the PDF rendering itself happens on their service.
  2. In Odoo, open Settings → PDF Generator API.
  3. Paste the credentials and click Test Connection.
  4. Tick the document types you want, save, and click Generate custom PDF on any matching record.

Privacy & data flow

When you generate a PDF, the record data mapped in your dataset is transmitted to pdfgeneratorapi.com over HTTPS — for an invoice, typically: number, dates, state, currency, company and customer details (name, address, VAT), line items (description, quantity, price, taxes), totals, payment reference, and notes.

Requests are signed with a short-lived JWT (HS256) using your API secret. No data leaves Odoo outside an explicit Generate action (or an async batch you dispatched).

See the PDF Generator API privacy notice (at pdfgeneratorapi.com/privacy-notice) for how transmitted data is handled.
