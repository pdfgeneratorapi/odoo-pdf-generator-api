Connector for Odoo Invoicing & Accounting

Replace Odoo's invoice PDF with your own design — no QWeb required

This connector plugs pdfgeneratorapi.com into your existing Odoo Invoicing app. Add a Generate custom PDF button to invoices, design templates with a visual editor, and the Send wizard automatically emails your custom PDF instead of the default Odoo report.

What it adds to Odoo

This is an extension, not a standalone app — it enhances Odoo's built-in Invoicing and Accounting modules with custom PDF rendering powered by pdfgeneratorapi.com. After install, you get:

  • A new Generate custom PDF split-button on the invoice form (next to Send and Print).
  • A Settings → PDF Generator API page to enter credentials and pick defaults.
  • Send-by-email substitution: when you click Send, customers receive the latest pdfgen PDF instead of Odoo's standard report.
  • Background batch generation with a webhook callback for hundreds of invoices at once.

Ships with a ready-to-use dataset for account.move (customer invoices, vendor bills, credit notes). The same machinery extends to any Odoo model via a per-model mapping wizard.

How it works

Connect your workspace

Paste your pdfgeneratorapi.com API key, secret, and workspace identifier in Settings → PDF Generator API. Click Test Connection to verify.

Design or import a template

Open the visual editor inside Odoo. Drag fields from the palette, style headers, add line tables, and preview against a real Odoo record.

Generate from any record

Click Generate custom PDF on an invoice (or any supported model). The PDF is attached and downloaded automatically.

Send to customers

The Send wizard picks the latest pdfgen PDF instead of the default report — no extra steps, no manual swap.

Configuration

  1. Sign up at pdfgeneratorapi.com and copy your API key, API secret, and workspace identifier (usually your account email).
  2. In Odoo, open Settings → PDF Generator API.
  3. Paste the credentials and click Test Connection.
  4. Open any invoice and click Generate custom PDF.

Privacy & data flow

When you generate a PDF, record data is transmitted to pdfgeneratorapi.com over HTTPS: invoice number, dates, state, currency, company and customer details (name, address, VAT), line items (description, quantity, price, taxes), totals, payment reference, and notes.

Requests are signed with a short-lived JWT (HS256) using your API secret. No data leaves Odoo outside an explicit Generate action.
